  • 基于行为树的 AI

    行为树

    示例

    僵尸 AI

    特点

    僵尸 AI 能根据自身情况（生命值，中枪与否）和外部条件（周围是否存在玩家），采取合理的行动。

    AI基础

    基本行为操控（靠近，远离，追逐，逃避…）

    寻路能力（从游戏场景中一个位置移动到另一个位置，最短路径）

    感知能力（自身状态如血量，听觉和视觉等感知能力）

    自主决策能力（根据自身状态和外部条件作出合理的反应）

    基本行为操控

    基本运动：行走，奔跑

    高级运动：跳跃，攀爬，躲避

    攻击：近身攻击，远程攻击，破坏障碍物

    寻路

    使用 Unity 的 Navigation 导航系统，实现自动寻路。

    感知能力

    视觉

    能够感知到玩家是否在视野范围内，视野内是否有障碍物

    听觉

    能够感知到玩家的脚步声，枪声，环境噪声（如爆炸，器械启动）

    受击反馈

    在受到伤害时（如玩家攻击，爆炸等），会有受击动作，同时能感知到伤害来源

    自主决策能力

    实现方式有：有限状态机，行为树，行为树与有限状态机结合

    有限状态机（Finite State Machine）

    稍微复杂的僵尸逻辑

    僵尸的感知范围内没有玩家时->僵尸随机游荡

    僵尸受到攻击->僵尸向开枪时所在的方向进行搜索

    僵尸感知到玩家->僵尸追踪玩家

    玩家进入僵尸攻击范围->僵尸攻击玩家

    玩家死亡或脱离僵尸感知范围->僵尸回到随机游荡

    僵尸生命值变为0->僵尸进入死亡状态

    类鸟群行为（集群行为）

    有时在游戏中为了添加沉浸感，会在深海环境，自然开放环境加入鸟群，鱼群。如何控制鸟群的运动，使其符合现实，不会突兀，且性能友好是该部分讨论的内容。

    参考视频：https://www.youtube.com/watch?v=bqtqltqcQhw

    国内视频翻译：https://www.bilibili.com/video/BV1VJ411y7wE/

    鸟群中每只鸟有以下三个规则：

    偏离附近的其他鸟（分离）

    迎合附近鸟群，转向同一方向（对齐）

    会自行向附近的鸟群中心飞去（凝聚）
